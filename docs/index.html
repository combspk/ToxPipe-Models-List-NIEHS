The following is a list of all AI models that are currently supported by the NIEHS deployment of ToxPipe. All models on this page are served via LiteLLM and are thus accessible using the Azure OpenAI schema. Supported subsets of these models are available via the LibreChat and ToxPipe API applications.

The model name in the “LiteLLM Internal Name” column is the exact way the name must be typed to correctly access the model via the LiteLLM API.

This list is subject to change as new models become available and old models are cycled out!
